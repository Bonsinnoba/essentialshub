# ElectroCom Storefront

The customer-facing application for the ElectroCom premium electronics store.

## Features
- **Dynamic Product Catalog**: Browse latest electronics with filtering and search.
- **Quick Shopping**: Floating "Add to Cart" and Wishlist buttons on product cards.
- **Secure Checkout**: Integrated with Paystack and Wallet systems.
- **Account Management**: Self-service profile updates and order tracking.
- **Responsive Design**: Optimized for desktop, tablet, and mobile with light/dark mode support.

## Development

### Setup
```bash
npm install
```

### Run Locally
```bash
npm run dev
```

### Build for Production
```bash
npm run build
```

The build artifacts will be located in the `dist/` directory.
